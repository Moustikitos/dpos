dposlib.ada

Cardano specific package trough Blockfrost provider.

You have to signup first and create a project. Token can be used on cardano testnet shelley or mainnet ada according to your project. blockfrost API can be requested using rest module.

>>> from dposlib import rest
>>> rest.use("shelley")
